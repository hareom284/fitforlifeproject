ရုံးမှာ အားတဲ့အချိန်လေး ဘာလို့ table tennis (ပင်ပေါင်) ကစားသင့်တာလဲ



ခုနောက်ပိုင်း အလုပ်ခွင်တွေဟာ တင်းကျပ်၊ တွန်းအားပေးတတ်တဲ့ အလေ့အထမရှိတဲ့ flexible workplace တွေဖြစ်လာကြတယ်နော်။ စည်းလွတ်ဝါးလွတ်ထားတားတာမဟုတ်ဘဲ စည်းကမ်းတကျလေး အလုပ်ခွင်ကိုကြပ်မတ်ထားပေမယ့် လှုပ်ရှားလုပ်ဆောင်မှုတွေကို လွတ်လပ်ခွင့်ပေးထားတာမျိုးပါ။ ဒီလိုအလုပ်ခွင်မျိုးမှာ ဝန်ထမ်းတွေက ပိုပြီး တက်ကြွတယ်၊ ဆန်းသစ်တဲ့ဖန်တီးနိုင်စွမ်းတွေရှိလာတယ်၊ အလုပ်အတွက်ရော၊ ကိုယ့်အတွက်မှာပါ တိုးတက်မှုရှိလာအောင် ကြိုးစားနိုင်လာတယ်၊ ကြောက်ရွံ့နေစရာမလိုတဲ့အတွက် ကိုယ်ပိုင်ဆုံးဖြတ်နိုင်စွမ်းရှိလာတယ်၊ စသည်ဖြင့် ကောင်းကျိုးတွေရှိတာပေါ့။ အဲဒီထဲကမှ တချို့ရုံးတွေမှာ ဝန်ထမ်းတွေရဲ့ခဏအနားယူချိန်(coffee break) မှာ table tennis ကစားဖို့ စီစဉ်ထားပေးတာတွေရှိပါတယ်။ table tennis ကစားရင်းနဲ့ ဘယ်လိုအကျိုးကျေးဇူးတွေပါ ရနိုင်လဲဆိုတာလေး စိတ်ဝင်စားဖို့ကောင်းပါတယ်။ ဆက်ကြည့်ရအောင် …

မျက်စိနဲ့ လက် ချိတ်ဆက်လှုပ်ရှားနိုင်စွမ်း ပိုကောင်းစေတယ်

စိတ်အာရုံ နိုးကြားမှု၊ အာရုံစူးစိုက်နိုင်စွမ်း ပိုကောင်းစေတာ နဲ့ မျက်စိလျင်၊ လက်မြန်စေတဲ့ အလေ့အကျင့်ပေါ့။ လူငယ်တွေအတွက်တော့ အလိုအလျောက်တုံ့ပြန်နိုင်စွမ်းကို ပိုသွက်စေတဲ့ကစားနည်းဖြစ်ပါတယ်။

ဖြတ်ထိုးဉာဏ် ပိုကောင်းစေတယ်

တင်းနစ်ဘောလုံးကို နေရာအချ၊ အဝှေ့အလှည့်နဲ့ ရိုက်တဲ့အရှိန်တို့က table tennis ကစားနည်းရဲ့အသက်ပါ။ ဒါတွေကြားမှာ ဉာဏ်သုံးပြီး လျင်မြန်မှုနဲ့ ထိန်းရင်း အနိုင်ကစားရတာလည်း ပညာပါတာပဲနော်။ table tennis ကျွမ်းကျင်လာသူတွေဟာ ပဟေဠိတွေ ဖန်တီးနိုင်စွမ်း၊ ဖြေရှင်းနိုင်စွမ်းမှာ ပိုထူးချွန်လာကြမှာပါ။

အဆစ်တွေကို မထိခိုက်စေတဲ့ ကိုယ်လက်လှုပ်ရှားမှုကိုရစေတယ်

ဒူး၊ ခါး၊ ခြေကျင်းဝတ်တို့ကို ဝန်သိပ်မပိစေဘဲ ကစားလို့ရတဲ့နည်းဖြစ်သလို တစ်ကိုယ်လုံးရဲ့လှုပ်ရှားမှုပါတာမို့ ကိုယ်လက်လှုပ်ရှားမှု ကောင်းကောင်းဖြစ်စေပါတယ်။

ကယ်လိုရီလောင်ကျွမ်းစေတယ်

ပေါင် ၁၅၀ လောက်ရှိတဲ့သူတစ်ယောက်ဟာ table tennis တစ်နာရီလောက်ကစားရင် ၂၇၂ ကယ်လိုရီလောင်ကျွမ်းစေနိုင်ပါတယ်။ စိတ်လည်းလန်းဆန်း၊ လူလည်းတက်ကြွလှုပ်ရှား၊ ပျော်စရာလည်း ကောင်း၊ အဆီလည်းကျစေမယ့် ကစားနည်းပါ။

ခင်မင်မှုတွေ တိုးစေတယ်

လုပ်ဖော်ကိုင်ဖက်အချင်းချင်း အလုပ်တွေကြောင့် တစ်နေ့လုံးနေလို့ တစ်ခါမှစကားမပြောဖြစ်ရင်တောင်မှ table tennis ကစားရင်း ရယ်မောမိ၊ ပြုံးရယ်မိကြမှာ သေချာတာပဲကို။ ကစားရင်းနဲ့ အချင်းချင်းခင်မင်ရင်းနှီးမှုတွေ ပိုတိုးလာနိုင်ပါတယ်။

ဟန်ချက်ထိန်းနိုင်စွမ်း ပိုကောင်းစေတယ်

ခန္ဓာကိုယ်ကို ဟန်ချက်ထိန်းပြီး လိုသလိုယိမ်းရ၊ ကစားရတာကလည်း table tennis မှာအရေးပါတာကြောင့် ဟန်ချက်ထိန်းနိုင်စွမ်းအတွက် ကွက်တိပါပဲ။

ဗျူဟာပိုကြွယ်စေတယ်

တစ်ဖက်လူရဲ့လှုပ်ရှားမှုကို မျက်ခြေမပြတ်သတိထားပြီး စက္ကန့်ပိုင်းအတွင်း ဗျူဟာကျကျကစားလိုက်နိုင်ဖို့ကလည်း လိုသေးတာကိုး။ ကိုယ့်ဦးနှောက်နဲ့ အာရုံနိုးကြားမှုကို လှုပ်ရှားတုံ့ပြန်မှုနဲ့ ချိတ်ဆက်တဲ့နေရာမှာ ဗျူဟာကျကျ အသုံးချနိုင်လာပါလိမ့်မယ်။ ကဲ… ကိုယ့်ရုံးမှာ table tennis မရှိသေးရင် ဒီဆောင်းပါးပြပြီး အခုပဲ တောင်းဆိုလိုက်ပါတော့လို့ မြှောက်ပေး(အဲလေ) အကြံပေးလိုက်ရပါတယ်။
